# dotfiles

## 各種ファイルの説明

| ファイル名    | 説明                                                                              |
| ------------- | --------------------------------------------------------------------------------- |
| .bash_profile | bash 初期化時に最初に呼ばれるスクリプト(読み込まれたら即時zsh を起動するだけです) |
| .envrc        | 環境変数を設定するためのファイルです                                              |
| .zshrc        | zsh 初期化時に最初に呼ばれるスクリプト(XDG_CONFIG_HOME/zsh/\*.zsh を読み込みます) |
