## メモ

- 目的のファイルへの到達性について
  - fzf でインタラクティブ検索でアクセス
    - ファイル名が分かってる場合に利用する
    - ファイルが存在するディレクトリ名が分かってる場合に利用する
  - fzf + ag でアクセス
    - テキストを検索してアクセスする
    - コードリーディングなどで利用
  - ヒストリからアクセス
  - バッファからアクセス
  - ファイラ
    - 目的のファイル名やディレクトリ名があいまいな場合に利用する
    - 目的のファイル名が分かっている場合も hjkl でファイルを選択できて楽
