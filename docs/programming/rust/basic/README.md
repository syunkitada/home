# Basic

## 学習方法

- まずは以下を読む

    - 基本的にこれだけ読めばよさそう
  - [ImosLab: Rust は何が新しいのか](https://imoz.jp/note/rust-functions.html)
    - 文法とかではなく言語仕様に関する話
    - メモリ管理の歴史と Rust のアプローチについて書かれている
