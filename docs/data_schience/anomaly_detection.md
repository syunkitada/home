# 異常検知

- ホテリング理論
- k 近傍法
  - 時系列に対するデータに対して距離を定め、それが異常値なのかどうかを判断する
  - 時系列に基づいたデータセットからある範囲のデータを切り取る
  - 異常値までの距離を計算し、距離の近さによって異常値かどうか判断する
- lof 法
